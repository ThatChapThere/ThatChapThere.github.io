Contribute Phylogeny

This project needs phylogeny filled in for every dinosaur. All help is gratefully recieved and will be credited under the name of your choice which can also link to your site/page on the credits page. You can do any taxon you like as long as it is monophyletic. Simply type out based on the following instructions, and then send directly in an email or in an attatched text file to ThatChapThere@outlook.com. If you have any more questions go to the FAQ page. The following method is the method you need to fill in taxa:

Choose a taxon

For this example we will choose Herrerasauridae.

Find a phylogeny

Google images or Wikipedia are good for this.


Find the nearest parent taxon on the project

In this case it is Saurischia.

Start the code

Put the name of the taxon you are filling in preceeded by an "@" symbol and beginning with a capital letter, within single quotes.
Add a space-colon-space " : ".
Put the name of the parent taxon of the taxon you are filling in preceeded by an "@" symbol and beginning with a capital letter, within single quotes.
End with a comma. 

	
'@Herrerasauridae' : '@Saurischia',

List all of the genera

Find all of the genera within the taxon and write youself a list. Wikipedia again is brilliant for this. On larger taxa there will be subtaxa listed and you'll have to click on those in order to be able to form a complete list of genera.

In this case we have:

Caseosaurus, Chindesaurus, Herrerasaurus, Sanjuansaurus and Staurikosaurus.

Choose a phylogeny

We will be using the Wikipedia one

Fill in your tree

Each taxon is listed simply as its name followed by the name of its parent taxon, with the quotes/colon/comma formation as described above/displayed below. Genera and species do not have an '@' symbol, while all other taxa do. Add a tab indent for each taxon level. 

	
'@Herrerasauridae' : '@Saurischia',
	'Staurikosaurus' : '@Herrerasauridae',

Unnamed tree sections

What about this point here?

These unnamed sections of the tree should have their names preceeded by a hash symbol "#". You can call them what you like, since their names won't be displayed, but preferably use something that makes sense. I prefer to use the name of a child taxon (in this case Herrerasaurus) followed by a "1": 

		
'@Herrerasauridae' : '@Saurischia',
	'Staurikosaurus' : '@Herrerasauridae',
	'#Herrerasaurus1' : '@Herrerasauridae',

If a an unnamed node has two children that are both unnamed nodes, then use the name of one of them but with a "2" instead of a "1" (or 3 or so on for even larger collections of unnamed nodes). The following is an example of this from another part of the tree: 

		
'#Aorun1' : '@Tyrannoraptora',
	'Aorun' : '#Aorun1',
		'zhaoi' : 'Aorun',
	'#Compsognathidae2' : '#Aorun1',
		'#Compsognathidae1' : '#Compsognathidae2',
			'@Compsognathidae' : '#Compsognathidae1',	
			'Aniksosaurus' : '#Compsognathidae1',
				'darwini' : 'Aniksosaurus',
		'@Maniraptoriformes' : '#Compsognathidae2',

If you can't quite make sense of this, just invent your own method. Just make sure to use hash symbols.

Fill in the rest of your tree

Remember, put the name of a taxon in single quotes, follow with a colon, then the name of its parent taxon in single quotes, then a comma. Each taxon should be indented one tab relative to its parent. 

		
'@Herrerasauridae' : '@Saurischia',
	'Staurikosaurus' : '@Herrerasauridae',
	'#Herrerasaurus1' : '@Herrerasauridae',
		'Herrerasaurus' : '#Herrerasaurus1',
		'#Chindesaurus1' : '#Herrerasaurus1'
			'Chindesaurus' : '#Chindesaurus1',
			'Sanjuansaurus' : '#Chindesaurus1',

Look up missing genera

In this case, Caseosaurus is unclassified. We can try to google "Caseosaurus phylogeny" or something like that.

The best I can find here is an incredibly blurred image with practically unreadable text that seems to indicate multiple possible classifications. I think we can safely say that Caseosaurus is incertae sedis (latin for "of uncertain placement").

To classify these we need to add a tree section that has its name preceeded by a hyphen "-", and that otherwise has the same name as its parent (minus the "@" symbol of course). 

		
'@Herrerasauridae' : '@Saurischia',
	'-Herrerasauridae' : '@Herrerasauridae',
	'Staurikosaurus' : '@Herrerasauridae',
	'#Herrerasaurus1' : '@Herrerasauridae',
		'Herrerasaurus' : '#Herrerasaurus1',
		'#Chindesaurus1' : '#Herrerasaurus1'
			'Chindesaurus' : '#Chindesaurus1',
			'Sanjuansaurus' : '#Chindesaurus1',

Then make the genus a child of this. 

		
'@Herrerasauridae' : '@Saurischia',
	'-Herrerasauridae' : '@Herrerasauridae',
		'Caseosaurus' : '-Herrerasauridae',
	'Staurikosaurus' : '@Herrerasauridae',
	'#Herrerasaurus1' : '@Herrerasauridae',
		'Herrerasaurus' : '#Herrerasaurus1',
		'#Chindesaurus1' : '#Herrerasaurus1'
			'Chindesaurus' : '#Chindesaurus1',
			'Sanjuansaurus' : '#Chindesaurus1',

Add species

Add the species in the same way as everything else, and make sure that their names begin with lowercase letters, while genera all begin with capitals. 

		
'@Herrerasauridae' : '@Saurischia',
	'-Herrerasauridae' : '@Herrerasauridae',
		'Caseosaurus' : '-Herrerasauridae',
			'crosbyensis' : 'Caseosaurus',
	'Staurikosaurus' : '@Herrerasauridae',
		 'pricei' : 'Staurikosaurus',
	'#Herrerasaurus1' : '@Herrerasauridae',
		'Herrerasaurus' : '#Herrerasaurus1',
			 'ischigualastensis' : 'Herrerasaurus',
		'#Chindesaurus1' : '#Herrerasaurus1'
			'Chindesaurus' : '#Chindesaurus1',
				'bryansmalli' : 'Chindesaurus',
			'Sanjuansaurus' : '#Chindesaurus1',	
				'gordilloi' : 'Sanjuansaurus',

Thats it!
