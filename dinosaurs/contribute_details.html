Contribute Details

The project needs details filling in for every dinosaur. All help is gratefully recieved. You can fill out whichever genera you like that are not listed as "described" on the reference. Preferrably do alphabetically consecutive dinosaurs, but you can do a taxon you like instead if you prefer. Simply type out based on the following instructions, and then send directly in an email or in an attatched .js or .txt file to ThatChapThere@outlook.com. If you have any more questions go to the FAQ page. The following are the details you need to fill in:

Name

Write the name of the dinosaur (beginning with a capital letter) within single quotes, followed by a colon and "new Genus(". 

	
	'Alaskacephale' : new Genus(72, 71, 2005, 4.5, [], 'Pachycephalosaurus', ['Alaska'], 'Sullivan', false, ''),

Time lived

Write the earliest possible time appeared and latest possible time extict in mya (millions of years ago), both followed by commas. One good place to find these details is dinochecker.com or Wikipedia. If Wikipedia does not have specific dates it will either mention a specific age e.g. the Maastrichtian (and the relevant dates should be mentioned on the specific age's Wikipedia page) or mention the dig site where the fossils were found (and the relevant dates should be mentioned on the dig site's Wikipedia page). Failing this, just go for the dates of a larger time period such as the middle Jurassic that you know for certain the genus did not exist outside of. 

	
	'Alaskacephale' : new Genus(72, 71, 2005, 4.5, [], 'Pachycephalosaurus', ['Alaska'], 'Sullivan', false, ''),

Year described

Write the year officially described not discovered, followed by a comma. 

	
	'Alaskacephale' : new Genus(72, 71, 2005, 4.5, [], 'Pachycephalosaurus', ['Alaska'], 'Sullivan', false, ''),

Length

Write the length in metres. Again, dinochecker.com and Wikipedia are good for this. If you can't find a length anywhere on the internet, find a paper listing bone lengths and scale it based on the bone lengths of its nearest relative. Failing that just make an educated guess based on the fossil sizes. Again, add a comma. 

	
	'Alaskacephale' : new Genus(72, 71, 2005, 4.5, [], 'Pachycephalosaurus', ['Alaska'], 'Sullivan', false, ''),

Junior Synonyms

Wikipedia should have these. Put each synonym in single quotes and put commas between them. Encase these within square brackets "[" and "]" and end with a comma. If there are no synonyms just use empty square brackets "[]". 

	
	'Alaskacephale' : new Genus(72, 71, 2005, 4.5, [], 'Pachycephalosaurus', ['Alaska'], 'Sullivan', false, ''),	
	
	'Allosaurus' : new Genus(156, 145, 1877, 12, ['Creosaurus', 'Labrosaurus', 'Antrodemus'], '', ['USA'], 'Marsh', false, 'Despite its popularity, Allosaurus has been considered potentially dubious because of its incomplete holotype.'),

Possible senior synonym

Wikipedia should have these too. Use single quotes and follow with a single comma. If the genus is definitely unique, just use empty quotes " ' ' " 

	
	'Alaskacephale' : new Genus(72, 71, 2005, 4.5, [], 'Pachycephalosaurus', ['Alaska'], 'Sullivan', false, ''),	
	
	'Allosaurus' : new Genus(156, 145, 1877, 12, ['Creosaurus', 'Labrosaurus', 'Antrodemus'], '', ['USA'], 'Marsh', false, 'Despite its popularity, Allosaurus has been considered potentially dubious because of its incomplete holotype.'),

Countries discovered in

Wikipedia should have these also. Put each country in single quotes and put commas between them. Encase these with square brackets "[" and "]" and end with a comma. Also important to remember is that Alaska and the USA (listed "USA") are considered separate countries for the sake of the map. 

	
	'Alaskacephale' : new Genus(72, 71, 2005, 4.5, [], 'Pachycephalosaurus', ['Alaska'], 'Sullivan', false, ''),	
	
	'Asiaceratops' : new Genus(100, 95, 1989, 1, [], '', ['Uzbekistan', 'China', 'Mongolia'], 'Lev Nesov, L.F. Kaznyshkina & Gennadiy Olegovich Cherepanov', true,  ''),

Desciber(s)

Wikipedia should have these as well. Put inside single quotes and end with a comma. always use "&", not "and". Use commas and spaces if there are three or more describers. Always put a period at the end of "et al.". 

	
	'Alaskacephale' : new Genus(72, 71, 2005, 4.5, [], 'Pachycephalosaurus', ['Alaska'], 'Sullivan', false, ''),	
	
	'Asiaceratops' : new Genus(100, 95, 1989, 1, [], '', ['Uzbekistan', 'China', 'Mongolia'], 'Lev Nesov, L.F. Kaznyshkina & Gennadiy Olegovich Cherepanov', true,  ''),	
	
	'Aorun' : new Genus(164, 161, 2013, 1, [], '', ['China'], 'Choiniere et al.', false, ''),

Filler "false" value

Type false without quotes and end with a comma. (This is a currently meaningless filler value, I apologise for my bad coding.) 

	
	'Alaskacephale' : new Genus(72, 71, 2005, 4.5, [], 'Pachycephalosaurus', ['Alaska'], 'Sullivan', false, ''),

Notes

Put any notes you have about the dinosaur in single quotes (or leave empty single quotes). Then finally end the line with a closing parenthesis and a comma "),". 

	
	'Alaskacephale' : new Genus(72, 71, 2005, 4.5, [], 'Pachycephalosaurus', ['Alaska'], 'Sullivan', false, ''),	
	
	'Allosaurus' : new Genus(156, 145, 1877, 12, ['Creosaurus', 'Labrosaurus', 'Antrodemus'], '', ['USA'], 'Marsh', false, 'Despite its popularity, Allosaurus has been considered potentially dubious because of its incomplete holotype.'),
